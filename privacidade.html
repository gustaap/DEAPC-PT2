Política de Privacidade

Esta Política de Privacidade explica como a LAVI Reservas trata os dados inseridos pelos utilizadores na plataforma.

1. Dados recolhidos

A plataforma pode guardar dados como nome, e-mail, password, histórico de sessão e informações de reserva. Nesta versão, esses dados são armazenados localmente no navegador através de localStorage ou sessionStorage.

2. Finalidade dos dados

Os dados são utilizados para permitir login, criação de conta, acesso ao perfil, simulação de reservas e diferenciação entre utilizador normal e administrador.

3. Armazenamento local

Como este projeto não utiliza backend real, os dados não são enviados para um servidor. Ficam apenas no navegador do utilizador.

4. Segurança

Esta versão é adequada para demonstração académica, mas não representa um sistema seguro para uso real. Numa aplicação real, passwords devem ser protegidas com hashing e processadas num servidor seguro.

5. Direitos do utilizador

O utilizador pode apagar os dados locais limpando o armazenamento do navegador ou terminando a sessão na plataforma.
